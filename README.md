1. Clone the repository
```bash
git clone https://github.com/yourusername/shoresquad.git

```

2. Initialize Git
```bash
git init
git add .
git commit -m "Initial commit: ShoreSquad website"
```

3. Open with Live Server
- Right-click `index.html` in VS Code
- Select "Open with Live Server"
- Site opens at `http://localhost:5500`
